## 📌 About

**VZN** is a full-stack news aggregation platform that automatically fetches, categorizes, and displays the latest stories from top global media outlets. Built with Django and deployed on Render, it brings together Tech, Gaming, Cinema, Anime, Science, and Comics news — all in one fast, clean interface.

> *"Your vision for news — curated, categorized, and always current."*

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔄 **Automated News Fetching** | Auto-pulls the latest stories from multiple global sources without manual updates |
| 🗂️ **Category Filtering** | Browse by Tech, Gaming, Cinema, Anime, Science, Comics and more |
| 📄 **Article Detail Pages** | Full story view with image, source attribution, and publish time |
| ⏱️ **Always Up-to-Date** | News is fetched automatically so readers always see the latest stories |
| 📱 **Responsive UI** | Fully optimized layout for desktop, tablet, and mobile devices |
| 🚀 **Live Deployment** | Production-deployed on Render, always accessible online |

## 📁 Project Structure

```
vzn-news/
│
├── VZN/                # Django project settings & configuration
├── home/               # Main app — views, models, URLs, templates
├── manage.py           # Django management script
├── requirements.txt    # Python dependencies
├── build.sh            # Render build script
├── render.yaml         # Render deployment configuration
├── runtime.txt         # Python runtime version
├── .gitignore
└── db.sqlite3          # SQLite database
```

---

## 🌐 Deployment

Deployed on **[Render](https://render.com)** using:

- `render.yaml` — defines the web service and environment
- `build.sh` — runs install, collectstatic, and migrate on each deploy
- `runtime.txt` — pins the Python version for the build environment
